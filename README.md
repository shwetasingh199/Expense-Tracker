# 💰 Expense Tracker App using Data Science

## 📌 Project Overview

This project is a **Data Science-based Expense Tracker App** that helps users analyze their spending patterns, track expenses, and gain financial insights through interactive visualizations.

---

## 🎯 Problem Statement

Managing daily expenses manually is difficult and often leads to overspending.
This project provides a **data-driven solution** to monitor, analyze, and optimize financial behavior.

---

## 💡 Solution

* Transformed raw financial data into structured format
* Performed data cleaning and feature engineering
* Conducted exploratory data analysis (EDA)
* Built an interactive dashboard using Streamlit

---

## 🚀 Features

✔ Expense tracking by category
✔ Monthly spending trends
✔ Category-wise analysis
✔ Interactive filters (Category, Month)
✔ KPI Metrics (Total Spending, Income, Savings)
✔ Visualizations (Bar, Pie, Line charts)

---

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Streamlit

## ⚙️ How to Run

### 1️⃣ Clone the repository

```bash
git clone https://github.com/shwetasingh199/Expense-Tracker
cd "C:\Users\shwet\EXPENSE TRACKER"
```

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run data pipeline

```bash
python main.py
```

### 4️⃣ Run dashboard

```bash
streamlit run dashboard/app.py
```
